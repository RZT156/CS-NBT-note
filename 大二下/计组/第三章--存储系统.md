# 3.1 概论
## 存储系统层次结构
分层次原因：
- 速度快的容量小，贵
- 便宜的速度慢容量大
分层次目的：
解决==存储容量、价格和存储速度==之间的矛盾
对存储器要求==容量大、速度快、成本低==
三级存储系统：cache+主存+辅存
内存=cache+主存
cache相对其他内存速度更快，内存更小。提升访问速度，以使存取速度和CPU运算速度相匹配
