binary search tree
二叉树 ->满足BST性质，结点值约束 -> 二叉排序树（二叉搜索树）BST
# 性质：左小，根中，右大
左子树上所有结点值均 < 根结点值；
右子树上所有结点值均 > 根结点值；
左、右子树本身又各是一棵二叉排序树。
# 特点：
- 二叉排序树的中序序列是一个递增有序序列；
![[BST's feature.png]]
# 插入：
1. BST为空，则创建一个`val`域为k的结点，将它作为根结点；
2. 否则将k和根结点的关键字比较，若两者相等，则说明树中已有此关键字k，无须插入，直接返回0；
3. 若`k < T->key`，则将k插入根结点的左子树中，否则将它插入右子树中。
# 删除：
（1）叶子节点可以直接删；
（2）只有左子树/右子树，结点替换
（3）完整的树，==这个很麻烦我看不懂==：以中序前趋替换，再删除该前趋；也可以用后继，方法一样
     ![[删除节点为完整树.png]]
# 查找：
很方便啊，时间O(log n)级的
查找失败：左边不满足：`target < current->val && current->leftchild==NULL` 或者右边不满足：`target > current->val && current->rightchild==NULL`
